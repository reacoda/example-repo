# Calculator App

The calculator app is a program that does simple calculations and then save them to a file. 

### Instructions for use
1. Firstly, clone the repository to your local repository.
2. Once done, you can navigate to the file calc_app.py.
3. You can open the file using your favourite code editor such as VSCode.
4. When you run the code, you will have the following options:
   - Perform a calculation.
   - View previous equations.
   - Exit.
